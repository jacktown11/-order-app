# order-app

使用`vue-clie3`仿制的饿了么外卖`WebApp`，这是一个`SPA`（单页面应用，`Single Page Application`）。主要技术包括`Vue.js`、`vue-router`、`Axios`，使用`better-scroll`插件优化页面滚动，`css`使用`stylus`编写，采用`flex`布局、`sticky footer`等技术。
